Pruebas de hipótesis¶

se siguen los siguientes pasos:

  1. Se formulan la hiptesis nula y la hiptesis alternativa.

  2. Se selecciona un estadstico tal que su distribucin muestral bajo la hiptesis nula sea conocida.

  3. Se seleccionan los datos.

  4. Se calcula el estadstico de los datos y se calcula su valor $p$ bajo la hiptesis nula.

  5. Si el valor $p$ es menor que la significancia predeterminada $\alpha$, se rechaza la hiptesis nula. Si el valor $p$ es mayor no se rechaza la hiptesis nula $H_0$.

Error Tipo I: rechazar la hiptesis nula cuando en realidad no debera rechazarse

Error Tipo II: dejar de rechazar la hiptesis nula cuando debera rechazarse

Generalmente el paso ms complicado en la lista que se ilustra es encontrar la distribucin muestral del estadtico de prueba.

Ejemplo:¶

considere una hiptesis nula que afirma que una variable aleatoria $X$ tiene una media $\mu_0 = 1$. Dadas las muestras de $X$, entonces deseamos probar si los datos muestreados son compatibles con la hiptesis nula. Aqu simulamos las muestras extrayendo $100$ muestras aleatorias de una distribucin ligeramente diferente a la que se afirma en la hiptesis nula (usando $\mu = 0.8$):

In [29]:
mu0, mu, sigma = 1.0, 0.8, 0.5
In [30]:
X = stats.norm(mu, sigma)
In [31]:
n=100
In [32]:
X_samples = X.rvs(n)

Dados los datos de muestra, " X_samples ", a continuacin debemos calcular un estadstico de prueba. Si se conoce la desviacin estndar de la poblacin $\sigma$, como en este ejemplo, se puede usar

$$ z = \frac{\bar{x}-\mu}{\sigma/\sqrt{n}} $$

el cual es distribuido normalmente.

In [33]:
z = (X_samples.mean() - mu0)/(sigma/np.sqrt(n))
z
Out[33]:
-4.180944153431165

Si no se conoce la varianza de la poblacin, se puede usar la desviacin estndar de la muestra en su lugar:

$$ t = \frac{\bar{x}-\mu}{\sigma_x/\sqrt{n}} $$

Pero en este caso el estadstico de prueba sigue una distribucin $t$ de student (ya no se distribuye normalmente).

Se calcula la desvicacin estndar de la muestra usando la funcin de numpy std con $ddof = 1$

In [34]:
t = (X_samples.mean() - mu0)/(X_samples.std(ddof=1)/np.sqrt(n))
t
Out[34]:
-4.472591761993936

En cualquier caso, obtenemos un estadstico de prueba que podemos comparar con la distribucin correspondiente para obtener un valor p.

Por ejemplo, para una distribucin normal, podemos usar una instancia de stats.norm para representar una variable aleatoria distribuida normal, y con su mtodo ppf, podemos buscar el valor estadstico que corresponde a un cierto nivel de significancia.

Para una prueba de hiptesis de dos lados el nivel de significancia 5% (2.5% en cada lado), el umbral estadstico es

In [35]:
stats.norm().ppf(0.025)
Out[35]:
-1.9599639845400545

Dado que las estadsticas observadas son aproximadamente -6.50, que es ms pequeo que el valor umbral 1.96 para una prueba de dos lados con un nivel de significacin del 5%, tenemos motivos suficientes para rechazar la hiptesis nula en este caso.
